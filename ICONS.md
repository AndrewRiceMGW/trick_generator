# Creating App Icons

For the PWA to work fully, you need to create app icons.

## Quick Option: Use a Generator

1. Go to https://favicon.io/favicon-generator/ or https://realfavicongenerator.net/
2. Upload an image or create text-based icon
3. Download the pack
4. Rename files to:
   - `icon-192.png` (192x192 pixels)
   - `icon-512.png` (512x512 pixels)
5. Place in the trick_generator folder

## Manual Option: Create Your Own

Use any image editor (GIMP, Photoshop, Canva, etc.):

1. Create a square image (512x512 pixels recommended)
2. Add a roller blade emoji 🛼 or skating icon
3. Export as PNG in two sizes:
   - 192x192 pixels → save as `icon-192.png`
   - 512x512 pixels → save as `icon-512.png`

## Design Tips

- Use a simple, recognizable icon
- Make sure it looks good on both light and dark backgrounds
- Keep text minimal (it needs to be readable at small sizes)
- Suggested content: 🛼 emoji or roller blade silhouette
- Background color: Use the app's purple gradient (#667eea)

## Temporary Workaround

If you don't have icons yet, the app will still work! The PWA will just use a default icon.
The core functionality is fully operational without custom icons.
